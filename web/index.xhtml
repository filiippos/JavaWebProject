About

Το Maths' Class είναι μία διαδικτυακή πλατφόρμα η οποία απευθύνεται σε μαθητές Γυμνασίου προκειμένου να ακονίσουν τις γνώσεις στους στα μαθηματικά και στις Αριθμητικές Παραστάσεις.Για τη δημιουργία του χρησιμοποιήθηκαν οι γλώσσες προγραμματισμού Java,Javascript,Php, ενώ χρησιμοποιήθηκαν διάφορες τεχνολογίες όπως Java Server Faces, Hibernate ,jQuery,ajax .

Σκοπος της Πλατφορμας

Η παρούσα web εφαρμογή κατασκευάστηκε στα πλαίσια του μαθήματος "Εκπαιδευτικό Λογισμικό" ως δεύτερη επιμέρους απαλλακτική εργασία. Σκοπός της εργασίας είναι η κατασκευή ενός εκπαιδευτικού λογισμικού εκπαίδευσης για Αριθμητικές παραστάσεις Μαθηματικών Γυμνασίου
